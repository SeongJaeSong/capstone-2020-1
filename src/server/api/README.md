### signup
[POST] /api/signup  
회원가입

요청
```
{
    "email": "~~@~~.~~"
    "pw": "qwer1234"
    "name": "홍길동"
    "age": 25
}
200 ok
400 요청 형식이 맞지 않음
409 이미 존재하는 계정
```
### login
[GET] /api/login  
로그인 만료여부 조회

요청
```
{
    "email": "~~@~~.~~"
    "uuid": "~~~~-~~~~-~~~~-~~~~"
}
```
응답
```
{
    "email": "~~@~~.~~"
    "expiry": "Sun, 03 May 2020 18:13:18 GMT",
    "uuid": "~~~~-~~~~-~~~~-~~~~"
}
200 ok
400 요청 형식이 맞지 않음
403 이미 만료돼서 재로그인 필요
404 검색 결과가 존재하지 않음
```
