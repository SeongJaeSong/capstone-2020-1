### login
[GET] /api/login  
아이디와 패스워드로 로그인

요청
```
{
    "email": "~~@~~.~~"
    "pw": "qwer1234"
}
```
응답
```
{
    "email": "~~@~~.~~"
    "pw": "qwer1234"
    "name": "홍길동",
    "age": "25",
}

404 존재하지 않는 계정 또는 pw틀림
```
